# PC端组件索引

> [!info] 说明
> 本项目使用 Element Plus 组件库，低开平台的组件文档为参考。
> `学习资料/低开组件文档/` 下有完整的官方组件文档。

## 项目使用的 Element Plus 组件

| 组件 | 用途 | 相关页面 |
|------|------|----------|
| `el-table` | 数据表格 | 所有列表页 |
| `el-form` | 表单 | 入库/出库/调拨表单 |
| `el-dialog` | 弹窗 | 详情/编辑弹窗 |
| `el-card` | 卡片容器 | 搜索区域/统计卡片 |
| `el-select` | 下拉选择 | 供应商/仓库/客户选择 |
| `el-input` | 输入框 | SN码录入/搜索 |
| `el-button` | 按钮 | 操作按钮 |
| `el-tag` | 标签 | 状态显示 |
| `el-pagination` | 分页 | 列表分页 |
| `el-date-picker` | 日期选择 | 入库/出库日期 |
| `el-descriptions` | 描述列表 | 详情展示 |

## 低开平台 PC 端组件对照

低开平台提供了以下组件类别的官方文档：

- **容器组件**: container, drawer, modal, page, row, steps, tabs
- **展示组件**: button, carousel, chart, icon, iframe, image, tag, text, tree
- **数据组件**: cardList, table
- **表单组件**: checkbox, datePicker, fileUpload, form, input, select, switch 等 20+
- **OA组件**: processForm, processTable

> [!tip] 开发参考
> 低开组件的事件、属性、适配器机制与 Element Plus 有差异。
> 我们通过 iframe 嵌入独立 Vue 项目，**不依赖低开组件系统**。
