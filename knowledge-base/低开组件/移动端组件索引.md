# 移动端组件索引

项目移动端使用 Vant 4 组件库。

## 常用 Vant 组件

| 组件 | 用途 |
|------|------|
| `van-form` | 表单 |
| `van-field` | 输入框 |
| `van-button` | 按钮 |
| `van-popup` | 弹出层 |
| `van-tag` | 标签 |
| `van-list` | 列表 |
| `van-cell` | 单元格 |
| `van-picker` | 选择器 |
| `van-dialog` | 对话框 |
| `van-toast` | 轻提示 |

## 已实现的移动端页面

- `src-mobile/views/Purchase/StockIn.vue` — 移动端采购入库
- `src-mobile/views/Sale/StockOut.vue` — 移动端销售出库
- `src-mobile/views/Transfer/TransferConfirm.vue` — 移动端调拨确认
- `src-mobile/views/Return/SaleReturn.vue` — 移动端退货
