# 字段名规范铁律

> [!danger] P0 铁律 — 第4次重复违规后固化
> 调用低开平台 API 时，**所有字段名必须以 `docs/MODEL_API_DOCS.md` 为准**。
> 接口文档来源于平台模型定义，是全项目唯一的字段名真相源。

## 转换链路

```
前端 camelCase  -->  request.js 转换 snake_case  -->  发送给低开平台
平台 snake_case  -->  request.js 转换 camelCase  -->  前端使用
```

> [!important] 关键理解
> 模型字段是 snake_case（如 `created_at`），request.js 自动转为 camelCase（`createdAt`）。
> 前端不要自创简写（如 `createTime`），必须严格对应模型的 snake_case 原名。
