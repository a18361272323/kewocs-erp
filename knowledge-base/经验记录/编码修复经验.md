# 编码修复经验

## 2026-05-29 Cloudflare 构建排障全程

### 症状1: 模块只编译21个 → 中文乱码

```
[vite:vue] Missing semicolon. (51:15)
```

**根因**: 之前用 Python `rb`/`wb` 编辑 .vue 文件时破坏了 UTF-8 多字节中文字符。

### 症状2: Element is missing end tag

```
src/views/Warehouse/Transfer.vue (32:9): Element is missing end tag
```

**根因**: `Out-File -NoNewline` 吞掉了所有换行符，文件变成一行。

### 症状3: Rollup failed to resolve import

```
Rollup failed to resolve import "/@api" from Dashboard.vue
```

**根因**: 路径写错，应为 `@/api`（Vite 别名）。

## 正确修复流程

> [!checklist] 标准修复步骤
> 1. `git show <commit>:<path>` 取原始干净文件
> 2. 用 PowerShell UTF-8 原生读写做修改
> 3. `[System.Text.UTF8Encoding]::new($false)` 无BOM写入
> 4. 全面扫描: `Get-Content -Raw | ? { $_ -match '\ufffd' }`
> 5. Push → 等 Cloudflare 构建验证

## 错误做法（避免）

- ❌ Python `open(file, 'rb')` / `open(file, 'wb')` 编辑 UTF-8 中文文件
- ❌ 猜测文件编码（GBK/UTF-16）后盲目转换
- ❌ `Out-File -NoNewline` 写 .vue 文件
