# 低开平台排障指南

## 模型常见故障

| 问题 | 排查点 |
|------|-------|
| 模型方法调用失败 | modelKey/methodKey 是否正确 |
| 入参不生效 | 参数名是否与 SQL #{xxx} 匹配 |
| 分页异常 | current/pageSize 是否传递 |
| 数据未更新 | 是否调用正确 methodKey (add vs edit) |

## 接口常见故障

| 问题 | 排查点 |
|------|-------|
| HTTP 接口不通 | 是否申请域白名单 |
| 返回格式异常 | 请求/响应适配器配置 |
| 业务编排报错 | 节点间变量传递 |

## 组件常见故障

| 类型 | 问题 | 排查点 |
|------|------|-------|
| 展示类 | iframe 不显示 | src 地址 / CSP 策略 |
| 展示类 | 图片不显示 | 防盗链配置 |
| 表单类 | 校验不生效 | 正则配置 |
| 表单类 | 下拉无数据 | 数据源绑定 |
| 表格类 | 数据不加载 | 适配器绑定 |
| 表格类 | 分页不工作 | total 字段映射 |

## 联动常见故障

| 问题 | 排查点 |
|------|-------|
| A 变化 B 不更新 | ctxData 传递路径 |
| 显隐控制失效 | 条件表达式语法 |
| updateStore 不生效 | 数据源名称 |
