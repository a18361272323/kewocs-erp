# 薪福通低代码平台概述

## 平台定位

薪福通（XFT）低代码开发平台，面向企业内部的快速应用搭建。
科沃斯ERP基于该平台的数据模型、接口开发和页面嵌入能力。

## 核心概念

- **应用** (`app`): 一个独立的业务系统，对应 `reg4bc6558503724`
- **模型** (`model`): 数据表定义，托管在平台后端数据库
- **模型方法** (`model_method`): 对模型的 CRUD 操作，通过 key 调用
- **接口** (`api`): HTTP 接口 / 业务编排 / SQL脚本节点
- **组件** (`component`): 页面 UI 构建块（Table、Form、Select 等）
- **变量系统**: `xcInput`、`xcOutput`、`ctxData`、`xcUser`

## 与我们项目的关系

> [!summary] 我们使用低开平台的方式
> - **数据层**: 21个托管模型 → `docs/MODEL_REFERENCE.md` 记录所有字段和 SQL
> - **API层**: 通过 `model_method` 的 key 调用 CRUD → `src/api/index.js`
> - **页面层**: 独立 Vue 3 项目嵌入 iframe，不依赖低开页面搭建
> - **账款层**: 调用薪福通账款管理模块的应收单 API
