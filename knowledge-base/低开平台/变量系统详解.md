# 低开平台变量系统

## 内置变量

| 变量 | 作用域 | 说明 |
|------|-------|------|
| `xcInput` | 接口请求适配 | 原始请求参数，只读 |
| `xcOutput` | 接口响应适配 | 最终返回数据，可写 |
| `xcHeader` | 接口上下文 | 读写请求头 |
| `xcQuery` | 接口上下文 | 读写 URL 查询参数 |
| `xcUser` | 全局 | 当前用户信息 (userId/userName/deptId) |
| `xcAuth` | 全局 | 用户权限信息 |
| `xcInitialize` | 应用初始化 | 初始化数据 |

## 页面变量

| 变量 | 说明 |
|------|------|
| `ctxData` | 组件间数据联动 |
| `ctxData.args` | 事件触发时传递的参数 |
| `updateStore` | 更新数据源 |

## 核心场景

### 请求适配 (xcInput -> xcOutput)
```javascript
// 驼峰转下划线
xcOutput = {
  order_no: xcInput.orderNo,
  customer_name: xcInput.customerName
}
```

### 组件联动 (ctxData)
```javascript
// Table 选中行 -> Form 回填
ctxData.formData = ctxData.args.row
```

### 组件显隐
```javascript
visible: ctxData.status === 'CONFIRMED'
```

## 适配器执行顺序
1. 页面加载时 (初始化)
2. 事件触发时 (onClick/onChange)
3. 数据变更时 (Watch)
