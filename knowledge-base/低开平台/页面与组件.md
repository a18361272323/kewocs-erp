# 低开平台页面与组件

> [!info] 我们的实现方式
> 使用 Element Plus + 独立 Vue 3 项目，通过 iframe srcdoc 嵌入。
> **不依赖低开组件系统**。以下为平台文档参考。

## 页面搭建

- **页面编辑器**: 拖拽式可视化编辑
- **主题样式**: 导航颜色、布局排版
- **导航入口**: PC 顶部导航+侧边菜单 / 移动端底部 Tab
- **事件系统**: 组件事件 + 触发器 + 适配器联动

## PC 端组件

### 容器
Page, Row, Tabs, Steps, Modal, Drawer

### 展示
Text, Image, Button, Icon, Tag, Chart, Carousel, Tree

### 数据
Table, CardList

### 表单
Form, Input, Select, DatePicker, FileUpload, RichText 等 20+

### OA
processForm, processTable

## 移动端组件

### 容器
Page, Grid, Tabs, Drawer, Dropdown

### 展示
Text, Image, Button, Icon, Tag, Chart, Carousel, Steps

### 数据
CardList

### 表单
Form, Input, Picker, Selector, DatePicker, FileUpload

## 自定义组件
- 支持上传第三方组件
- 通过 iframe 嵌入独立页面 (我们的模式)
