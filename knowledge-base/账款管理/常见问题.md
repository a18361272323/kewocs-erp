# 账款管理常见问题

## 开通与权限

> [!question] 如何开通账款管理？
> 白名单制，联系招行客户经理开通。超级管理员到应用中心启用并分配权限。

## 开票相关

> [!question] 如何提交开票申请？
> 业务员在应收账单/销售合同中点击"申请开票"，确认金额后提交。

> [!question] 开错发票怎么办？
> 发起红字确认单 -> 财务审核 -> 重新开票。

> [!question] 开票申请被退回怎么办？
> 查看退回原因 -> 修改后重新提交。

## 报错排查

| 报错信息 | 处理方式 |
|---------|---------|
| 存在相同的开票申请单，有重复开票风险 | 检查是否已提交过相同单据 |
| 商品编码与商品名称不匹配 | 检查商品信息是否完整 |
| 请检查单据的确认状态、审批状态 | 确认单据是否已审批 |
| 银行流水无该账户 | 检查云直联授权状态 |

## 数据权限

> [!question] 财务想看所有单据？
> 在通用权限管理 -> 数据权限中配置。
