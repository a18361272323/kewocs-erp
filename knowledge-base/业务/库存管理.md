# 库存管理

## 调拨流程

```mermaid
flowchart LR
    A[选择调出仓库] --> B[加载仓库INSTOCK的SN]
    B --> C[勾选要调拨的SN]
    C --> D[选择调入仓库]
    D --> E[保存调拨单]
    E --> F[确认调拨]
    F --> G[逐条更新SN仓库归属]
```

### 关键点
- **SN级调拨**（非品类级）：勾选具体SN
- 确认后逐条更新每个SN的 `warehouseId` 和 `warehouseName`
- 涉及模型: 调拨单 (MOIrlRmiFH) + SN码 (MOk2ZJ4aga)

## 盘点流程

```mermaid
flowchart LR
    A[选择盘点仓库] --> B[自动加载INSTOCK的SN]
    B --> C[扫描实盘SN码]
    C --> D{比对}
    D -->|正常| E[标记正常]
    D -->|盘亏| F[标记LOST]
    D -->|盘盈| G[标记盘盈]
    E --> H[完成盘点]
    F --> H
    G --> H
```

### 关键点
- 选择仓库 -> 自动加载 INSTOCK 的SN列表
- 扫描实盘SN -> 系统比对
- 盘亏SN: 自动标记为 LOST
- 盘盈SN: 记录额外信息

## 库存台账

- 模型: 库存台账 (MOsWdYRJhQ)
- 提供低库存预警 (`getLowStockCount`)
- 仓库汇总 (`getWarehouseSummary`)
- 预警列表 (`getAlertList`)
