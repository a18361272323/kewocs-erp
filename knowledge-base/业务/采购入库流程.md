# 采购入库流程

## 流程图

```mermaid
flowchart TD
    S[选择供应商+仓库] --> SN[录入SN码]
    SN --> P[选择商品+单价]
    P --> ADD[添加到入库明细]
    ADD --> CHECK{继续录入?}
    CHECK -->|是| SN
    CHECK -->|否| SUBMIT[提交入库]
    SUBMIT --> CONFIRM[确认入库]
    CONFIRM --> UP[更新SN状态为INSTOCK]
    UP --> FINISH[完成]
```

## 关键步骤

### 1. 录入SN码
- 扫码枪或手动输入SN码
- 选择对应商品和入库单价
- 可连续录入多条

### 2. 提交入库
- 创建入库单 (add)
- 批量更新SN码状态为 INSTOCK
- 设置仓库归属

### 3. 确认入库
- 更新入库单状态为 CONFIRMED
- 可推送应付单（账款管理）
